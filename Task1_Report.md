# Containerization Assignment

## Containerization
Containerization packages applications and dependencies into lightweight containers.

## Use Cases
- Microservices
- CI/CD
- Cloud deployments
- Scalable web applications

## Docker, Podman, Kubernetes
Docker: container platform.
Podman: daemonless alternative.
Kubernetes: orchestration platform.

## Difference from VMs
Containers share the host OS kernel while VMs run a full guest OS.

## Kubernetes Role
Provides scaling, load balancing, self-healing and orchestration.
